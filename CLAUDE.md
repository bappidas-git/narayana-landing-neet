# Narayana NEET Coaching — Google Ads Landing Page

## Project Overview
This is a high-converting Google Ads landing page for Narayana Coaching Centers' NEET (Medical) coaching programme, targeting students and parents in Guwahati, Assam, and North-East India.

## Cloned From
This codebase is cloned from the IIT-JEE landing page (`narayana-landing-iit`). The structure, components, styling, and all functionality are identical. Only the CONTENT has been updated for NEET.

## Tech Stack
- React 18 (CRA) with lazy loading & Suspense
- Material UI v5 with custom theme
- Framer Motion for scroll-triggered animations
- CSS Modules for scoped styling
- Swiper for carousels
- SweetAlert2 (via swalHelper.js) for alerts
- Iconify for icons
- Pabbly webhook integration (via webhookSubmit.js)

## Key Files to Update (Content Only)
- src/data/ — All data files (courses, highlights, benefits, centre, courseDetails)
- src/components/sections/ — Section JSX files (text content only)
- src/components/common/Header/ — Nav labels if different

- public/index.html — Meta tags, schema markup, loader tagline
